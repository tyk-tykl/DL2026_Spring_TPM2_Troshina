# Smart Delivery Prediction — PM/BA Case (Wildberries)

Продуктовый кейс по улучшению пользовательского опыта Wildberries через внедрение AI-функции прогнозирования доставки.

## Ключевая идея

**Проблема**  
Непредсказуемая доставка снижает доверие пользователей и напрямую влияет на CR и retention.

**Решение**  
AI-функция **Smart Delivery Prediction** — прогноз доставки с диапазоном и вероятностью, обновляемый в реальном времени.

**Эффект**
- CR: +2–5%
- Retention (30 дней): +3–7%
- Обращения в поддержку: −10–20%
- Точность ETA: ≥ 80%

**Цель:** превратить доставку из «лотереи» в предсказуемый сервис.

## Контекст

Wildberries — лидер по ассортименту и доступности, но уступает конкурентам в предсказуемости доставки.  
Это создает ключевой риск — **потерю доверия пользователей**.

## MVP (3 месяца)

**Что делаем:**
- Прогноз диапазона доставки (min–max)
- Вероятность доставки (%)
- Отображение в ключевых точках (карточка, корзина, checkout, трекинг)
- A/B-тест

**Что НЕ делаем:**
- Сложные ML-модели  
- Персонализацию  
- Внешние данные (погода, трафик)

MVP сознательно жертвует точностью ради скорости запуска и проверки гипотезы.

## Ключевые риски

- Неточный прогноз → потеря доверия  
- Недостаток данных  
- Сложность интеграции  

## Итог

Проект решает ключевую проблему Wildberries — **непредсказуемость доставки**,  
которая напрямую влияет на доверие и повторные покупки.

Быстрый запуск MVP позволяет:
- проверить влияние на бизнес-метрики  
- снизить пользовательский негатив  
- заложить основу для AI-driven логистики  

**Главный результат:** повышение доверия пользователей и рост повторных покупок.
